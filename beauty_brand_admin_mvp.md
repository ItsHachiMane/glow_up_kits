# Beauty Brand Admin MVP

## Goal
Build a small internal admin web app for the makeup kit business using Python + Django.

## MVP scope
- Manage products and kits
- Track inventory
- View and update orders
- See a simple sales summary
- Keep customer notes

## Core screens
1. Dashboard
   - total sales
   - open orders
   - low stock alerts

2. Products
   - add/edit/delete products
   - name, price, cost, photo, notes

3. Inventory
   - stock on hand
   - reorder threshold
   - restock amount

4. Orders
   - order list
   - order status
   - customer name
   - items

5. Customers
   - customer list
   - order history
   - notes

## Suggested Django models
- Product
- InventoryItem
- Order
- OrderItem
- Customer
- Note
